# 📝 Interactive To-Do List App

A clean, responsive, and light-weight To-Do List web application built with modern HTML, CSS, and vanilla JavaScript. It allows users to add tasks, mark them as completed, delete tasks, and automatically saves your list locally so data isn't lost on page reload.

---

## ✨ Features

* **Add Tasks:** Quick task creation via input field and 'Add' button.
* **Toggle Status:** Click on any task to mark it as completed (with line-through and checked icon).
* **Delete Tasks:** Easily delete individual tasks using the cross (`×`) button.
* **Data Persistence:** Integrated with `localStorage` to keep your tasks saved in the browser.
* **Responsive Design:** Styled with a clean gradient background and centered UI for mobile and desktop screens.

---

## 🛠️ Tech Stack

* **HTML5** - Structure and semantics
* **CSS3** - Styling, responsive layout, dynamic custom checkboxes
* **JavaScript (ES6)** - DOM manipulation and Local Storage integration
